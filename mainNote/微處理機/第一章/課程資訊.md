# 課程資訊

## 課程基本資料

- **課程名稱**：微處理機 (Microprocessors)
- **教科書**：Sepehr Naimi 等作者關於 STM32F103 的書籍
- **學習平台**：iLearning

## 課程大綱

本課程涵蓋以下主要章節：
1. ARM 介紹
2. 組合語言基礎
3. 定址模式
4. C 語言程式設計
5. GPIO 操作
6. UART 通訊
7. Timer 計時器
8. Interrupt 中斷處理

## 評量方式

課程評量包括：
- 出席率
- 作業
- 期中考
- 期末考
- 實習報告
