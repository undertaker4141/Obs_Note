# 硬體介紹 (STM32F103)

## STM32F103 "Blue Pill" 開發板

STM32F103 開發板（俗稱 "Blue Pill"）是一款基於 ARM Cortex-M3 核心的微控制器開發板，因其藍色 PCB 而得名。這款開發板價格低廉、功能強大，是學習 ARM 微控制器的理想平台。

### 主要特點
- **處理器**：STM32F103C8T6 (ARM Cortex-M3 核心)
- **時脈速度**：最高 72 MHz
- **Flash 記憶體**：64 KB
- **SRAM**：20 KB
- **供電**：5V 或 3.3V
- **尺寸**：小巧，約 5.3 x 2.2 cm

### 接腳配置

Blue Pill 開發板提供了豐富的 I/O 接腳，包括：
- 多個 GPIO 接腳
- UART 接腳
- SPI 接腳
- I2C 接腳
- ADC 接腳
- 定時器接腳

## STM32F103 內部架構

STM32F103 的內部方塊圖顯示了其主要組件和連接方式：

### 核心組件

- **Cortex-M3 CPU 核心**：32 位元 RISC 處理器，支援 Thumb-2 指令集
- **記憶體**：
  - Flash 記憶體（程式碼儲存）
  - SRAM（資料儲存）
- **時脈系統**：
  - 內部 RC 振盪器
  - 外部晶振支援
  - PLL 倍頻器
- **除錯介面**：SWD (Serial Wire Debug)

### 週邊設備

STM32F103 包含豐富的內建週邊：
- **GPIO**：通用輸入/輸出接腳
- **UART/USART**：串列通訊介面
- **SPI**：串列週邊介面
- **I2C**：雙線串列匯流排
- **ADC**：類比數位轉換器
- **定時器**：多種定時器，包括基本、通用和進階定時器
- **DMA**：直接記憶體存取控制器
- **RTC**：實時時鐘
