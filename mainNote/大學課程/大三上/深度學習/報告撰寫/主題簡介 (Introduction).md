## 1. 主題簡介 (Introduction)

在攝影過程中，透過玻璃窗、展示櫃或車窗等透明介質進行拍攝時，光線的反射往往會與透射光線重疊，導致成像品質下降，產生干擾視覺的偽影。從物理光學模型來看，相機所捕捉到的混合影像 $I$ 可視為透射層 $T$（玻璃後的真實場景）與反射層 $R$（玻璃表面的反射場景）的線性疊加，數學上通常表示為 $I = T + R$（部分模型亦考慮衰減係數）。單一影像去反光（Single Image Reflection Removal, SIRR）的目標即是僅從單張輸入影像 $I$ 中恢復出清晰的背景層 $T$。由於在缺乏額外深度資訊、偏振光或多視角約束的情況下，將單一影像分解為兩個分量存在無限多種數學解，因此這在本質上是一個高度不適定（Ill-posed）的逆問題。

早期的研究多依賴基於優化（Optimization-based）的方法，利用手工設計的先驗知識（Hand-crafted Priors），如梯度稀疏性（Gradient Sparsity）、相對平滑度（Relative Smoothness）或鬼影線索（Ghosting Cues）來進行圖層分離。然而，這些方法在處理複雜且多變的真實場景時，往往因先驗假設失效而導致泛化能力不足。近年來，深度學習技術的引入徹底改變了此領域，透過卷積神經網絡（CNNs）從大規模數據中學習映射關係已成為主流。

儘管早期的深度學習模型（如 ERRNet [1]）已取得顯著進展，但仍面臨諸多挑戰，例如合成數據與真實數據之間的領域差異（Domain Gap）、特徵提取過程中的語義流失，以及難以區分強反射與背景等問題。隨著技術演進，研究焦點已從單純的 CNN 轉向更先進的架構設計：引入 Transformer 以捕捉長距離特徵（如 DSIT [4]）、利用可逆網絡（Reversible Network）減少資訊流失（如 RDNet [5]），以及結合模型驅動優化的深度展開網絡（Deep Unfolding Network，如 DExNet [6]）。

在本專題中，我們旨在重新檢視並評估 SIRR 領域的發展脈絡與最新突破。我們選用了六個具代表性的模型進行實驗，涵蓋了從早期的特徵增強方法 [1]、知識蒸餾架構 [2]，到最新的位置感知框架 [3]、雙流 Transformer [4]、可逆解耦 [5] 及深度展開技術 [6]。為了確保比較的公平性與嚴謹性，我們依據課程規範構建了統一的訓練與測試數據集，並在 Google Cloud Platform (GCP) 的高效能運算環境下，對所有模型進行了完整的重新訓練（Retraining）。本報告將詳述各方法的理論基礎、實作細節，並透過七個基準數據集上的定量指標與定性視覺結果，深入分析各模型在真實世界去反光任務中的優劣。
